Actividad R_2: Cálculo del Error en Robot Móvil Diferencial
Equipo: Robonigg
Descripción:
Este proyecto implementa un sistema de localización y monitoreo de errores para el robot diferencial Puzzlebot utilizando ROS 2 Humble. El objetivo principal es estimar la pose actual ($x, y, \theta$) mediante odometría (integración de Euler) y calcular en tiempo real la distancia euclidiana y el error angular respecto a una meta definida.
